# CyberShield - مشروع خدمات الأمن السيبراني

### فكرة المشروع:
موقع ويب تعريفي متكامل ومتجاوب لمهندس خدمات أمن سيبراني وتقنية معلومات، يحتوي على لوحة تحكم ذكية لعرض إحصائيات وهمية مباشرة للتهديدات، ونموذج تواصل آمن ومحمي.

### الأدوات المستخدمة:
* **HTML5:** لبناء الهيكل الدلالي للموقع.
* **CSS3:** لتنسيق الواجهات بالنظام الداكن (Dark Mode).
* **JavaScript:** لإضافة تفاعل العداد المباشر والتحقق من صحة مدخلات النموذج (Form Validation).
